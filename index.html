Documentación de la Aplicación de Carpintería de Aluminio CORTAL

Descripción General

La Aplicación de Carpintería de Aluminio es una herramienta en línea que ayuda a calcular los cortes necesarios para diferentes aberturas y medidas en trabajos de carpintería de aluminio. Esta aplicación permite a los usuarios ingresar dimensiones y configuraciones específicas para obtener resultados precisos.

Requisitos Previos

Navegador web moderno compatible (Chrome, Firefox, etc.).
Conexión a Internet para acceder a la aplicación en línea.


Instalación y Configuración

No se requiere instalación, ya que la aplicación es accesible a través de un navegador web.
Asegúrese de tener una conexión a Internet estable.
Abra su navegador web y vaya a la URL de la aplicación.

Guía de Uso

Página de inicio:

Al abrir la aplicación, se mostrará la página de inicio que muestra los diferentes tipos de aberturas que se pueden calcular.
Haga clic en el botón del tipo de abertura que desea calcular para acceder a la interfaz de dicha opción.

Interfaces Ventanas, Monoblocks, Puertas y Mamparas:

Las interfaces le permiten ingresar las dimensiones y configuraciones para calcular los cortes.
Complete los campos requeridos, como el ancho, alto, tipo de aluminio, etc.
Haga clic en el botón "Calcular" para obtener los resultados.
Cada campo contiene validaciones según lo que se requiera, por lo que, si se ingresa algún dato incorrecto o se omite, la aplicación no avanzará hasta que se ingrese el dato correcto.

Resultados:

Después de hacer clic en "Calcular", se mostrarán los resultados de los cortes necesarios en un ticket que aparecerá en la parte derecha de la pantalla y que incluye una cruz en la parte superior derecha que al hacer clic cierra el ticket.
El ticket de resultados incluirá información detallada, como las dimensiones de los cortes, materiales requeridos, medidas para vidrios, etc.
Puede guardar o imprimir los resultados haciendo clic en el botón de la impresora que aparece junto con el Ticket.

Atajos.

Haciendo clic en el logo en la barra lateral izquierda se vuelve al inicio.

API y Funcionalidades

La aplicación no proporciona una API para interactuar con ella.

Personalización y Configuración

La aplicación no admite personalización o configuración adicional.

Consideraciones de Seguridad

La aplicación no maneja datos sensibles ni requiere medidas de seguridad adicionales.

Solución de Problemas y Preguntas Frecuentes

La aplicación no carga correctamente:
Asegúrese de tener una conexión a Internet estable.
Intente borrar la caché del navegador o utilizar otro navegador web.

Los resultados no son precisos:
Verifique que haya ingresado las dimensiones y configuraciones correctamente.
Si encuentra un error persistente, comuníquese con nuestro equipo de soporte.

Para preguntas, comentarios o problemas reportados, contáctenos en andaristis@gmail.com o utilice el formulario de contacto en nuestro sitio web.
